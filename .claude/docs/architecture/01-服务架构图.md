## 5. 系统边界

### 外部依赖

| 依赖 | 用途 | 备选方案 |
|------|------|----------|
| DeepSeek API | LLM 分析与审核 | Qwen / OpenAI（通过 `--provider` 切换） |
| GitHub API v3 | 仓库搜索采集 | 无需认证（限额 60/h），带 Token（5000/h） |
| RSS Feed URLs | RSS 内容采集 | 12 个源，可开关（`rss_sources.yaml`） |

### 内部接口

| 接口 | 协议 | 用途 |
|------|------|------|
| MCP Server | JSON-RPC 2.0 over stdio | 知识库检索（search / get / stats） |
| CLI | argparse | 流水线执行入口 |
| OpenCode Agent | IDE 集成 | `@collector / @analyzer / @organizer` 调用 |
